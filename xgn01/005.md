#### word

today  英[ təˈdeɪ ]  美[ təˈde ] 今天

```
I'm late today

birthday [ ˈbɜ:θdeɪ ] 也叫 B-day
Today is my birthday! 主系表
```

well [ wel ] 身体好的

- adv. 好；很；好意地；高高兴兴地；
- adj. 健康的；井的；良好的；恰当的；

```
good 好

good / well 区别
good 更加广泛的好
well 通常指身体的好
fine 也是表达好，天气好，心情好


what's wrong 怎么了
I'm fine,Really 我很好，真的
```

see [ si: ] 看到，看见

```
I see a dog

I know / I see 好的我明白了 
I see 更加地道

see you / see you next time 下次再见
```
